## Why
Boston AI Tinkerers events are rich but noisy: attendees need help deciding which sessions to catch, who to meet, and how to turn event details into an actual plan.

## What Changes
- Build a Boston AI Tinkerers event scout app with seeded meetup data.
- Add AI-powered session-fit recommendations and attendee planning output.
- Package it as a single-page app deployable to Cloud Run and publishable on Sundai.

## Capabilities
- New capability: `meetup-scout`
- New capability: `session-fit`
- New capability: `attendee-planning`
