# Boston AI Tinkerers Scout

AI meetup scout for Boston AI Tinkerers. Explore upcoming sessions, describe your goals, and generate a personalized event plan with session recommendations, networking targets, and prep questions.

## Run locally

```bash
bun install
bun run dev
```

## Environment

- `OPENROUTER_API_KEY`
- `OPENROUTER_MODEL`
- `OPENROUTER_BASE_URL` (defaults to `https://openrouter.ai/api/v1`)
